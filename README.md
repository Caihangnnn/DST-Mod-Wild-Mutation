# 荒野异化 Wild Mutation

《饥荒联机版》服务端模组。随机强化世界中的生物和 Boss，让战斗随天数推进变得更有变化和挑战。

Steam 创意工坊订阅链接：[Wild Mutation](https://steamcommunity.com/sharedfiles/filedetails/?id=3733170876)

## 主要功能

- 普通生物有概率获得随机异化词缀。
- Boss 会获得更高强度的异化效果。
- 支持精英怪、体型变化、伤害减免、吸血、再生、毒、冰冻、噩梦等效果。
- 可按世界天数逐步提高异化强度，并提供开局保护期。
- 可选择是否在生物名称中显示异化词缀。

## 配置项

- 普通怪强化倍率
- 普通怪异化概率
- Boss 强化倍率
- 精英怪出现概率
- 控制效果开关
- 体型变化开关
- 伤害减免开关
- 开局保护天数
- 词缀名称显示开关
- 天数成长曲线开关
- 疯狂模式开关



## 兼容说明

模组会修改生物的生命、伤害、速度、体型、名称和部分攻击效果。若其他模组也修改同类内容，可能出现数值叠加或显示冲突。
